How I do my computing

I'll talk here about the tools I use in my daily life. Nobody asked me to do this, but I'd like to talk about all this somewhere.

Hardware

Keyboard

I use the Keychron K6 mechanical keyboard as my daily driver, it has a very pleasant typing in a perfect layout. I definitely recommend it to those who like 65% keyboards.

Computer

I use the DELL Inspiron 3593 (didn't find a link corresponding to my product) as my daily driver.

Specifications:

  • 15.6" screen
  • Intel i5 10th gen.
  • 8GB of RAM
  • 256GB SSD

Software

  • Operating System: GNU/Linux.
  • Distribution: Gentoo.
  • Window Manager: i3-gaps.
    • Status Bar: i3blocks.
    • Dmenu: rofi.
    • Display Manager: SDDM.
  • Programming Languages:
    • Scripting languages: BaSH and Perl.
    • Main language: Rust.
  • Text/Code editor: Vim.
  • Shell: Fish.
  • Terminal Emulator: Alacritty.
