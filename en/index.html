Wanager - 2020
Wanager is a package and projects manager for the C programming language written in Rust It makes possible easy and fast projects and header files creation. It also permises to compile and launch a project without having to write a 80 chars long gcc comand or to hoard a makefile
It is also able to install libraries or to launch tests units.
