Wanager - 2020
Wanager is a package manager and build tool for the C programming language written in Rust. It makes possible the project and header creation faster and easier. Just build and run (without having to write a 80 chars long gcc command or to hoard a Makefile).
It allows you to rapidly run tests & to install libraries and packages
