WNG - 2020
WNG is the W package manager, a package manager and project manager for the C programming language.
It allows you to easily create projects and header files, to run tests suites, to compile big amounts of code and to install libraries for either github, gitlab and bitbucket.

Wlvm - 2020
Wlvm is a simple stack based virtual machine written in Rust. It has its own assembly like language with around 20 possible instructions.
REPO
Orion - 2021
Orion is an interpreted lisp inspired programming language. It is statically and strongly typed and allows efficient scripting by its tiny but complete std and its easy syntax.
