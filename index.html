Wanager - 2020
Wanager is a package manager and build tool for the C programming language written in Rust. It permises the fast project & header creation, easy build & run (without having to write a 80 chars long gcc command or to hoard a Makefile).
It allows you to rapidly run tests & to install libraries and packages
